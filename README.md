# AutoSamplerSorter

  Sort Mainstage 3 auto-samples for use in Ableton Live

  [![Build Status][travis-image]][travis-url]

Install
-------
stable
```bash
npm install -g auto-sampler-sorter
```
edge
```bash
npm install -g https://github.com/daxxog/auto-sampler-sorter/tarball/master
```
